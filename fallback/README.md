# Fallback — live-demo insurance

Drop the pre-recorded demo and screenshots here so a slow or flaky CI run can't
derail the talk.

Suggested contents (record during the dry-run in `DEMO_SETUP.md` step 6):

- `demo.mp4` (or `.gif`) — screen capture of: comment `@claude fix this` →
  Actions run → PR opens → CI green. Trim to ~60–90s.
- `pr.png` — screenshot of the final PR: the diff, the new test, Claude's summary.
- `actions-log.png` — screenshot of the workflow run log (for the "under the hood" beat).

During the talk: if the live run stalls past ~90s, cut to `demo.mp4` and keep moving.
